ALX React App

A basic React application created with **Vite** as part of the ALX Frontend ReactJS program.

- React
- Vite
- JavaScript
- Node.js & npm

## Setup & Run

```bash
git clone <your-repo-url>
cd alx-fe-reactjs/alx-react-app
npm install
npm run dev
